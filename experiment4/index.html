Experiment 4 - Infinite Worlds

Description

What are you trying to acomplish with this experiment?

For this experiment, we are trying to create a simple text generator similar to a Mab Libs. This one is supposed to generate an online recipt similar to one you would find from Amazon.

Technical

Describe in detail what you did technically and how it works?

I added in JS and functionality for the buttons with flags to set the buttons hidden. After clicking the buttons, they should generate the result with either the default option or the Fisher yates option with you either choosing the top of the options or the bottom of it.

Reflection

Each person that worked on the code should reflect on the process, the difficulties, and the successes of the experiment.

This project was easy going but I did have difficulties with trying to get the buttons to work as I haven't done buttons in javascript and I had to make sure that the buttons were hidden when clicking the other ones which took a while to get the correct flags. I found the algorithm for the Fisher-Yates Shuffle to be interesting and I wished that there would more shuffling algorithms we could look at besides just this one. Another difficulty was to get the spacing of the lines correct and I had to search stackoverflow for the solution as it had to with JQuery and how it dealt with text.
